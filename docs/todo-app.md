# TODO アプリメモ

## 目的
最低限の TODO アプリを Waku/React で動かすための実装メモです。ホーム (`/`) に配置している `TodoApp` コンポーネントではシンプルに「追加・完了トグル・削除」ができます。

## 使い方
1. `npm run dev` で開発サーバーを起動し、表示された URL にアクセスします。
2. 入力欄にタスクを記入して `Add` を押すと一覧の先頭に追加されます。
3. タスク行をクリックすると完了/未完了が切り替わります。右端の `×` を押すと削除されます。

## 技術的なポイント
- `src/components/todo-app.tsx` は `'use client'` なローカル state だけで構成し、要件通り `useEffect` は使用していません。
- UI の責務は `TodoForm`（入力・送信）, `TodoList`（一覧表示）, `todo-types.ts`（共有型定義）へ分割し、状態管理ロジックと描画を分離しました。
- タスク ID には `Date.now()` を使い、ブラウザリロードまでの一意性を担保しています。
- 入力欄は空白のみでは追加できず、残タスク数を表示することで現在の状態がわかります。
