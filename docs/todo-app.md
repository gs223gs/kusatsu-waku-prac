# TODO アプリメモ

## 目的
TODO アプリを Waku/React で動かすための実装メモです。ホーム (`/`) に配置している `TodoApp` は「追加・編集・完了トグル・削除」に加えて、人のアサイン・期限・優先度を管理できます。

## 使い方
1. `npm run dev` で開発サーバーを起動し、表示された URL にアクセスします。
2. タスク名を入力し、必要に応じてアサイン担当者・期限・優先度を設定して `Add task` を押すと一覧の先頭に追加されます。
3. タスク行をクリックすると完了/未完了が切り替わります。`Edit` からは既存タスクの内容を修正でき、`Delete` で削除できます。

## 技術的なポイント
- `src/components/todo-app.tsx` は `'use client'` なローカル state だけで構成し、要件通り `useEffect` は使用していません。
- UI の責務は `TodoForm`（入力・送信）, `TodoList`（一覧 + 編集 UI）, `todo-types.ts`（共有型定義 + 初期値）へ分割し、状態管理ロジックと描画を分離しました。
- 編集時は `TodoList` 内でドラフト state を管理し、保存ボタンから親の `onUpdate` を呼んで単一ソースでデータを更新します。
- タスク ID には `Date.now()` を使い、ブラウザリロードまでの一意性を担保しています。
- 入力欄は空白のみでは追加できず、残タスク数を表示することで現在の状態がわかります。
