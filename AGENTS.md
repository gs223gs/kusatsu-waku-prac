## 概要
React フレームワーク 「waku」 を練習するためのリポジトリ

@waku.readme.md にwakuの説明が書いてある

## 要件
簡単なTODOアプリ

### 追加要件1
- 編集機能
- 人のアサイン
- 期限設定
- 優先度設定

### 追加要件2
- カテゴリ設定
- タグ設定
- メモ設定

### 追加要件3
- アサインする人の固定化 => アサインする人を事前に入力 その人をトグルメニューで選択できるようにする
- タグの固定化 => タグを事前に入力したものをトグルメニューで選択できるようにする
- カテゴリの固定化 => カテゴリを事前に入力したものをトグルメニューで選択できるようにする

### 追加要件4
- タグの追加フォームの作成
- タグの複数選択

### 追加要件5
- カテゴリの追加フォームの作成
- カテゴリの複数選択

### 追加要件6
- アサインする人の追加フォームの作成
- アサインする人の複数選択

## 禁止事項
- 曖昧な状態でのコーディング禁止 必ず答えを出してからやること
- useEffect の禁止
